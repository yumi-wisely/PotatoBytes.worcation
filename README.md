# 🥔 PotatoPlanning (by PotatoBytes)
旅をホクホク快適にする旅程管理ツール。

## ✨ 主な機能
- **My Trip**: 複数の旅をカード形式で管理
- **Hybrid Timeline**: 全員の予定と個人の予定をスイッチして確認
- **Potato ID**: Googleログインで自分専用のID（QRコード）を発行
- **Integrated Map & Chat**: 場所の確認も相談もこれ一つで

## 🛠 技術スタック
- React / Tailwind CSS
- Lucide React (Icons)
- Firebase (Auth / Firestore)
